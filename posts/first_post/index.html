如何搭建一个静态页面博客

我用的是macOS，仅做参考使用

1.安装hugo

使用homebrew安装即可，当然也可以使用源码进行安装

brew install hugo

验证一下安装情况

hugo version
hugo v0.85.0+extended darwin/amd64 BuildDate=unknown

2.使用Hugo创建博客

hugo安装成功后，使用hugo new site创建博客站点

hugo new site blog

这个命令创建一个为名blog的目录，这是博客的根目录。目录结构如下：

blog
├── archetypes
│   └── default.md
├── config.toml         # 博客站点的配置文件
├── content             # 博客文章所在目录
├── data                
├── layouts             # 网站布局
├── static              # 一些静态内容
└── themes              # 博客主题

3.开始写博客的准备工作

首先，Hugo站点没有默认指定主题，所以我们要先选择一个主题，Hugo官方的主题网站。我选择的是Loveit这个主题。 关于主题相关的更多消息可以在Loveit的页面中查看。 然后我们要对主题进行安装，为了方便博客的管理，使用git进行版本和内容的控制。 初始化git。

git init

初始化完毕之后，我们对于主题进行下载，使用以下命令.

git submodule add https://github.com/dillonzq/LoveIt.git themes/LoveIt

等待git拉去代码完毕，我们的主题就安装完毕了。接下来是对于博客相关的配置，即对于config.toml进行基本的配置，以及相关主题内容的配置。配置引导参考这篇博客。

4.创建第一篇博文

配置完毕之后接下来就是书写第一篇博文了。

4.1 创建博客的markdown文件

打开终端/控制台，运行下面指令。

hugo new posts/first_post.md

hugo会在blog根目录下面创建一个\content\posts的目录，并含有first_post.md文件。

4.2 编辑内容

采用markdown语法编辑博客内容，具体实现和一些选项的设置可以参考这篇博客。

4.3 本地预览

写完之后想要知道最终呈现的效果，那么我们就要在本地运行起来看一下最后的呈现效果。使用以下命令启动网站。

hugo serve

查看http://localhost:1313/

4.4 部署到github

提交项目并利用github action自动部署，这个内容不少我可能会另外写一篇博客来记录相关操作。

5. 总结

至此，博客就搭建完毕了，开始记录我的各种值得记录的东西。希望对他人也能有所帮助。
